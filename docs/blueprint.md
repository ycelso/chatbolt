# **App Name**: EchoFlow

## Core Features:

- Chatbot Interface: Interfaz de usuario para mostrar la interfaz del chatbot.
- Session Management: Generar ID de sesión únicos para cada conversación.
- Webhook Integration: Enviar el ID de sesión y los mensajes del usuario al webhook de N8N: https://n8n-4oyr.onrender.com/webhook/gpt.
- Response Display: Mostrar las respuestas recibidas del webhook de N8N.

## Style Guidelines:

- Color primario: Gris claro (#D3D3D3) para una sensación limpia y moderna.
- Color de fondo: Blanco hueso (#FAFAFA) para proporcionar un contraste sutil y una experiencia de lectura cómoda.
- Color de acento: Gris oscuro (#4A4A4A) para el texto y los elementos interactivos para garantizar la legibilidad y la concentración.
- Fuente del cuerpo y del encabezado: 'Inter' (sans-serif) para una apariencia moderna y neutra adecuada tanto para los encabezados como para el texto del cuerpo.
- Usar iconos minimalistas en gris oscuro para mantener un diseño limpio y consistente.
- Usar un diseño limpio y simple con una jerarquía visual clara para facilitar la navegación y la legibilidad.
